# electron-spark
## 环境需求
Windows  
[nodejs 20+](https://nodejs.cn/download/)
## 使用方法
- 克隆项目到本地
```shell
git clone https://github.com/GentsunCheng/spark-control.git && cd spark-control
```
- 安装node模块
```shell
npm install -g yarn
yarn init
yarn install
```
- 直接运行
```shell
yarn start
```
- 构建
```shell
yarn build
```
构建好的包在项目的 **dist** 目录下
